# Objectives and Scope

## Objectives
- Learn basic security assessment techniques
- Identify vulnerabilities using scanning tools
- Understand how attackers exploit weaknesses
- Perform risk assessment using CVSS scoring
- Improve documentation and reporting skills

## Scope
- Testing limited to an intentionally vulnerable virtual machine
- Network scanning, vulnerability scanning, and basic exploitation included
- No testing on live or production systems
- No denial-of-service or social engineering attacks performed
