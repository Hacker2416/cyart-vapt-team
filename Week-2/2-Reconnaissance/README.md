Task 2: Reconnaissance Practice

Objective:
The objective of this task is to perform reconnaissance and OSINT (Open-Source Intelligence)
to gather publicly available information about a target domain.

Target:
- Domain: amazon.com
- Reconnaissance Type: Passive (OSINT)
- Environment: Academic / Educational

Tools Used:
- WHOIS
- Sublist3r
- Wappalyzer
- Shodan
- Maltego

Outcome:
Information related to domain registration, subdomains, technologies,
exposed services, and asset relationships was successfully identified
using publicly available sources.
